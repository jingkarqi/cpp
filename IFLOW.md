# C++ 数据结构实验项目

## 项目概述

这是一个用于南京林业大学经济管理学院"数据结构与应用算法"课程的实验项目。项目主要用于生成和管理实验报告，并实现各种数据结构算法的演示代码。

### 主要组成部分

1. **实验报告生成系统**：使用Node.js和Python脚本生成标准化的实验报告Word文档
2. **C++算法实现**：包含各种数据结构算法的C++实现代码
3. **报告模板**：提供Markdown和Word格式的实验报告模板

## 技术栈

- **编程语言**：C++, JavaScript, Python
- **文档处理**：docx库（JavaScript和Python版本）
- **Node.js依赖**：docx v9.5.1
- **Python依赖**：python-docx

## 项目结构

```
E:\cpp\
├── 实验报告.md              # 实验报告Markdown模板

```

## 主要功能

### 1. 实验报告生成

#### 使用Node.js生成报告
```bash
node create-report.js
```
该脚本会生成一个标准的实验报告Word文档，包含：
- 学校和课程信息
- 实验内容和要求说明
- 实验步骤和结果记录表格
- 指导教师签名区域

#### 使用Python注入内容
```bash
python inject.py          # 注入算法实现和伪代码
python inject_fill.py     # 填充实验内容
python inject_format.py   # 格式化报告
```

### 2. C++算法实现

项目包含各种数据结构算法的C++实现，目前已实现：
- 顺序表插入算法（ListInsert_Sq）

#### 编译和运行C++代码
```bash
g++ -o ListInsert_Sq.exe ListInsert_Sq.cpp
./ListInsert_Sq.exe
```

## 实验报告内容结构

每个实验报告应包含以下内容：
1. **实验名称**：具体的算法或数据结构名称
2. **实验时间**：进行实验的日期
3. **实验地点**：实验环境
4. **实验内容**：简要描述实验目标和内容
5. **实验步骤**：
   - 算法思路
   - 实现过程
   - 代码结构
6. **实验结果**：
   - 程序输出
   - 结果分析
7. **源代码**：完整的C++实现

## 开发工作流程

1. **创建新的算法实现**：
   - 编写C++源代码文件
   - 编译并测试算法实现
   - 记录实验结果

2. **生成实验报告**：
   - 运行`create-report.js`生成基础报告模板
   - 使用Python脚本填充具体实验内容
   - 根据需要调整报告格式

3. **报告管理**：
   - 每个实验应保存独立的报告文档
   - 源代码文件应与报告内容对应
   - 保持代码注释清晰，便于理解

## 注意事项

1. **代码规范**：
   - 使用有意义的变量名
   - 添加适当的注释
   - 保持代码结构清晰

2. **实验要求**：
   - 需要实现不少于5个教材中的算法
   - 每个算法都需要有完整的实验报告
   - 报告应包含伪代码和实际C++实现

3. **文件管理**：
   - C++源文件名应体现算法功能
   - 实验报告文件应包含实验名称和日期
   - 避免覆盖重要的实验文档

## 扩展指南

如需添加新的算法实现：

1. 创建新的C++源文件
2. 实现算法逻辑
3. 编写对应的实验内容文本
4. 使用提供的Python脚本生成或更新实验报告
5. 测试并验证算法正确性

项目已提供了完整的实验报告生成和管理工具链，可高效支持数据结构课程的多项实验任务。
